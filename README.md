# Oxford Hack 2018

## Getting started

### Word similarity service (backed by spaCy)
Runs on port 5000.
```
$ cd src/word_similarity

########################################
# Only include during first time setup
$ python3 -m venv venv
########################################

$ source venv/bin/activate
$ pip install -r requirements.txt
$ python3 similarity.py
```

## Webapp
Runs on port 3000.
```
$ npm install
$ npm start
```
